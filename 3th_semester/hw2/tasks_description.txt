1. Написать программу.

Со стандартного потока ввода задаётся вещественное число ε.

Для ввода использовать функцию scanf :

double x;

scanf("%lf",&x); // &x — это адрес переменной x, в которую нужно записать считанное число

Далее в цикле до признака конца файла вводится вещественное число x.

EOF со стандартного ввода можно получить нажатием Ctrl-D .
 

Для введённого x вычисляется и печатается значение √x.

Для вычисления квадратного корня используется итеративная формула

X0 = 1, X(i+1) = (Xi +X/Xi)/2 .

Вычисления продолжаются, пока |Xi − X(i+1) | >= ε. Ответом является числоX(i+1) .

 

Каждый ответ должен размещаться на отдельной строке.

Ответ напечатайте, используя функцию printf, в нескольких вариантах : с помощью форматных преобразований %f, %g, %e, %.10g и т. п. (см. описание )

 

Стандартные математические функции использовать нельзя.

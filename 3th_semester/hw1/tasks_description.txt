Написать функцию для подсчета количества натуральных чисел n (N1<=n<= N2), которые являются палиндромами (т.е. одинаково читаются справа налево и слева направо). 
Каждый палиндром нужно вывести на печать, а их количество вернуть как результат функции. N1 и N2 передаются как параметры.

     

В main в цикле считывать со стандартного ввода N1 и N2.

Закончить работу при вводе нуля. Если N1>N2 , выдавать диагностику. Для нормальных данных вызывать вашу функцию.
Для ввода использовать функцию scanf :
int x;
scanf("%d",&x); // &x — это адрес переменной x, в которую нужно записать считанное число
